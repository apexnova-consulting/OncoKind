# PWA Icons

Add these files for "Add to Home Screen" and install prompts to work:

- **icon-192.png** — 192×192 px (home screen icon on Android; used by some launchers)
- **icon-512.png** — 512×512 px (splash / high-res icon)

## Quick way to generate

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator — upload a 512×512 logo and download the pack; copy `icon-192.png` and `icon-512.png` here.
2. Or export 192 and 512 versions from your design tool and name them exactly as above.

Without these files, the app may still be installable on some devices, but icons will fall back to browser default or be missing.
